# Legato

Online learning platform inspired by [SolidSail](https://www.solidsail.com/)

## Description

This project was created to practice Tailwind CSS, Vite, Firebase and TypeScript

## Getting Started

### Dependencies

- Node.js 14.18+, 16+

### Installing

```shell
git clone https://github.com/esogelola/Legato.git
cd to the repo
```

### Executing program

```shell
npm install
npm run dev
```
